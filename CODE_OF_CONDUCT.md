# Code of Conduct

DaySynth のコミュニティでは、相互に敬意を払い、安心して参加できる場を維持します。

## 期待する行動

- 相手の立場や経験差を尊重すること
- 建設的で具体的なフィードバックを行うこと
- 事実と提案を分けて話すこと
- 迷ったら前提を確認し、勝手に断定しないこと

## 許容しない行動

- 侮辱、嫌がらせ、差別的な発言
- 秘密情報や個人情報の晒し行為
- 議論の場を壊す攻撃的なふるまい
- 他者の貢献を不当に否定する行為

## 運用

問題のある行動を見つけた場合は、まず記録を残し、必要に応じてメンテナーへ共有してください。重大なケースでは、関係者の安全を優先して対応します。

## 範囲

この規範は、issue、PR、レビューコメント、チャット、ドキュメント、その他プロジェクトに関するあらゆるやり取りに適用されます。
