# PWA（Progressive Web App）仕様書（廃止）

## 1. 状態

- 2026年2月2日に PWA 対応を廃止しました。
- 現在は通常の Web 導線での利用を前提とし、PWA 関連の機能・資産は保持していません。

## 2. 廃止理由

- Web から PWA への情報引き継ぎができず、運用面のメリットが限定的だったため。
- PWA への遷移を強制できず、通常の Web 導線と二重管理になっていたため。
- オフライン対応の維持コストに対して実利用が少なかったため。

## 3. 現在の導線

- 履歴・お気に入り・URL/ID から開く導線は `/history` に統合。
- 旧 `/home` は `/history` へ恒久リダイレクト。
- オフライン対応・Service Worker は未実装。

## 4. 変更履歴

- next-pwa / manifest / offline ページ / PWA バナーを削除。
- 既存の PWA ホーム画面機能を Web 導線へ統合。

---

本仕様書は PWA 機能の廃止方針を記録する目的で残しています。
